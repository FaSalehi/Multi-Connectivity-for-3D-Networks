# Multi-Connectivity for 3D Networks
This repository contains the source code for both the journal version and conference version of the papers:
1. "Reliability and Delay Analysis of 3-Dimensional Networks With Multi-Connectivity: Satellite, HAPs, and Cellular Communications", by Fateme Salehi, Mustafa Ozger, and Cicek Cavdar, published in IEEE Transactions on Network and Service Management. [doi:10.1109/TNSM.2023.3307909](https://ieeexplore.ieee.org/abstract/document/10227357)
2. "Ultra-Reliable Low-Latency Communication for Aerial Vehicles via Multi-Connectivity", by Fateme Salehi, Mustafa Ozger, Naaser Neda, and Cicek Cavdar, presented in EuCNC/6G Summit 2022. [doi: 10.1109/EuCNC/6GSummit54941.2022.9815803](https://ieeexplore.ieee.org/abstract/document/9815803)

The work conducts a multi-connectivity framework across aerial and terrestrial links to provide ubiquitous and reliable service to aerial vehicles. 
The code performs a Monte Carlo simulation to analyze end-to-end reliability and delay across different communication paths. It also proposes an optimal multi-connectivity solution for remote piloting of aerial vehicles to maximize spectral efficiency under stringent QoS constraints, including reliability, delay, and network availability.
