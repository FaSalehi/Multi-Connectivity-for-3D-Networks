# MC-for-3DNetworks

This repository contains the source code for the paper "Reliability and Delay Analysis of 3-Dimensional Networks With Multi-Connectivity: Satellite, HAPs, and Cellular Communications", by Fateme Salehi, Mustafa Ozger, and Cicek Cavdar, published in IEEE Transactions on Network and Service Management. (doi:10.1109/TNSM.2023.3307909)
